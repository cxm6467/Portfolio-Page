Temperature Conversion

This ruby program will prompt the user for a temperature in degrees Celsius and let the user know what the corresponding temperature is in Fahrenheit. 

				
## Ask for user input, the Farenheit value
puts "Please enter a value to convert FROM Celsius to Fahrenheit."

## Get that value and store it in a variable 
## (Chomping because of new lines and converting to a fixnum)
## Store input in variable to pass to method
to_conv = gets.chomp.to_i

## method to convert input n to output farenheit temp
def convert(n)
  return ((n * 1.8) + 32) 
end
## Do some math and print to the screen the result
puts "The temperature is #{convert(to_conv)} in Fahrenheit."

Foobar

This ruby program is an adaptation of a classic technical interview problem which displays sequences of the Foobar pattern. 

		  	
## Ask user for number
puts "Please enter a number:  "

##  Get that number and chomp it
number = gets.chomp.to_i

## Method to test the number whether its divisible by 3, 5, both
## or none
def foobar(number)
  ## iterate through /number/ times where is is the current iteration
  i = 1
  while i <= number
    if ((i % 3 == 0)&&(i % 5 == 0))
      puts("foobar")
    elsif(i % 3 == 0)
      puts("foo")
    elsif(i % 5 == 0)
      puts("bar")
    else
      puts("#{i}")
    end
    i = i + 1
  end
end
## process the result
foobar(number)
